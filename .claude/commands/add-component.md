Add a new HeroUI component to the project:

1. Check if the component package is already installed
2. If not, install it from @heroui/[component-name]
3. Add the component to tailwind.config.js content array
4. Create a usage example

Ask the user which component they want to add (e.g., modal, input, dropdown, etc.)
